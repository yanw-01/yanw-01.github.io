201922202246-闫威-计科20.1-实验3

山东大学 计算机科学与技术学院 云计算技术 课程实验报告
学号：201922202246 班 级： 计科20.1 姓名： 闫威 实验日期： 2023.3.14
实验题目：利用云平台搭建个人博客
实验学时：2
实验目的：熟悉个人博客系统的搭建。 具体包括： 参考方案：注册Github账号，搭建Hexo环境并实现个人博客搭建，撰写实验报告。
硬件环境： 联网的计算机一台
软件环境： Windows
实验步骤与内容：
安装node.js 安装git

安装hexo
1. 进入cmd窗口输入指令 npm install -g hexo-cli

检查安装是否成功

新建文件夹myblog并用git bash打开

初始化hexo 测试hexo s

打开http://localhost:4000/

将Hexo部署到Github 1. Github创建个人仓库

2. 生成ssh到github




进行部署
1.修改_config.yml文件

2. 安装deploy-git |

3.依次执行hexo c,hexo g,hexo d


4.回到github仓库

5.根据yanw-01.github.io访问博客

6.将实验报告上传到个人博客主页上
a.首先在 _posts文件夹下创建实验报告的md文件

b.清除Hexo博客的缓存，重新生成静态文件，并将hexo博客上传到远程服务器


c.打开个人博客
结论分析与体会：
通过这次实验，我成功地注册了Github账号，搭建了Hexo环境，并且实现了个人博客的搭建。在这个过程中，我学习了如何使用Git进行版本控制，了解了Hexo的基本概念，以及如何将hexo部署到github。
在整个实验的过程中，我发现搭建个人博客需要花费一定的时间和精力。需要掌握一些基本的技术知识，如Git版本控制、Markdown语言、Hexo搭建等。此外，还需要具备一定的耐心和毅力，因为在搭建的过程中会遇到一些问题，需要进行调试和解决，通过上网查找解决办法得以解决。
体会： 通过搭建个人博客，我不仅学会了如何使用Git版本控制、Markdown语言和Hexo搭建等技术，体会到了云计算技术带来的方便。
